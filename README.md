# Indra Automation Lab Website

A simple static website for presenting automation services and small Windows productivity tools.

## Pages

- `index.html` - main website
- `styles.css` - site styling
- `assets/automation-hero.png` - hero image

## View Locally

Open `index.html` in a browser.

## Publish With GitHub Pages

1. Open the GitHub repository.
2. Go to Settings.
3. Open Pages.
4. Choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save.

GitHub will give you a public website URL after it deploys.
